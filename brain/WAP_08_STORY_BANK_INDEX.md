## What This Doc Is

The master index for Nico's Story Bank. Individual story texts live in `brain/stories/` folder, one file per story (S001.md, S002.md, etc.). This index is the lookup table the Copywriting agent reads FIRST to identify which stories to pull.

This is not a collection of stories to be reproduced whole. It is raw material. The agent picks fragments — a line, an observation, a punchline, a stat — and weaves them into blog posts, YouTube scripts, Instagram captions, newsletters. Entire stories get told whole only occasionally (a dedicated YouTube story, a cornerstone blog post).

## Why Split Into Files

One story per file scales. Individual edits don't conflict. The agent loads only the stories it needs, not the whole bank. The GitHub UI stays readable.

## How Agents Use This Index

1. Identify the post topic and its themes (safety, food, driving, history, etc.)
2. Scan the Master Index below, filtering by Tags, Themes, or Use Cases
3. Read the index row fully — it contains Pullable Hooks, Core Observations, Pairs With references, and Named Business Flags
4. Open the linked story file (stories/SXXX_filename.md) for the full text
5. Pull fragments (hooks, observations, punchlines) into the draft
6. Never invent stories. If no match exists, flag [STORY NEEDED: topic] for Nico

## Critical Rules

- **Preserve Nico's exact phrases.** "Horizontal Italian mambo" does not become "Italian romance." "Sistine Chapel of pizza" does not become "mass of bad pizza." The specific phrases ARE the voice.
- **Respect Named Business Flags.** Story files may contain real business names (e.g. Battiloro restaurant) for authenticity. The `Named businesses` field flags these. In free content, swap named references for generic ones ("a restaurant near my apartment"). In paid content (The Sicilian Way, The Restaurant Guide), the full name may be used. See WAP_09 for the canonical paid-content directory.
- **Use Mode B for heavy topics.** Mafia victims, real tragedy, serious history — voice shifts per WAP_05.
- **Verify historical and statistical claims before publishing.** The `Stats to verify` field flags any numbers that need confirmation against primary sources.

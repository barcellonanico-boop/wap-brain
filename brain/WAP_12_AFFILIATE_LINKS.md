## What This Doc Is

The canonical registry of every affiliate link used across wearepalermo.com content.

When SOP_01 (Update Existing Post) or SOP_02 (Create New Post) needs to insert a hotel recommendation, tour link, or rental car link, the Copywriting or Architect agent looks HERE. Not the Booking partner dashboard. Not Google. Not memory. Here.

Every entry contains the canonical affiliate URL, plus enough context for Copywriting to decide which hotel/tour/etc. to recommend for a given post.

## Critical Rules

- **Use links from this doc only.** Never invent a Booking URL. Never modify an `aid` parameter. If a hotel is NOT in this doc, it does not get recommended — flag it for Nico to add.
- **Respect the verification date.** If `Last Verified` is older than 90 days, the link should be re-tested before publishing. SOP_05 (Affiliate Link Verification — to be built as Task 2.9) handles monthly re-verification.
- **Never mix with Direct Bookings.** The "Direct Bookings" section at the bottom is Nico's own property, not affiliate. Different economics. Do not treat as affiliate.
- **All Booking.com URLs must preserve `aid=918822`** as the affiliate parameter. This is Nico's partner ID. No `aid`, no commission.

## How Copywriting Uses This Doc

1. Identify the post topic (e.g., "where to stay in Palermo historic center")
2. Filter the registry by area (e.g., "Centro Storico")
3. Read "Nico's Take" + "Ideal For" to pick 2-4 hotels that match the post's angle
4. Insert the full affiliate URL (never shorten, never alter)
5. Follow WAP_06 affiliate disclosure rules in the post
